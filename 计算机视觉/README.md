# 模拟静电场智能测绘与分析系统 - 项目结构说明

## 项目概述
本项目是一个基于计算机视觉和AI技术的静电场测绘与分析系统，支持摄像头标定、数据采集、电场图生成和AI辅助分析。

## 项目结构

```
计算机视觉/
├── main.py                          # 主程序入口
├── ui/                              # UI模块包
│   ├── __init__.py                 # UI包初始化文件
│   └── ui_manager.py               # UI管理器，负责构建和管理GUI界面
├── business/                        # 业务逻辑模块包
│   ├── __init__.py                 # 业务逻辑包初始化文件
│   ├── business_manager.py         # 业务逻辑管理器，协调各业务组件
│   ├── cv_tracker.py               # 计算机视觉追踪器，处理探针标记检测
│   ├── calibration.py              # 标定模块，负责红色十字标定板检测
│   └── data_engine.py              # 数据分析引擎，处理电场数据和分析
├── hardware/                        # 硬件模块包
│   ├── __init__.py                 # 硬件包初始化文件
│   ├── hw_camera.py                # 摄像头管理器，处理摄像头采集
│   └── hw_serial.py                # 串口管理器，处理与硬件设备的通信
├── edit_data_engine.py             # 数据引擎编辑脚本（保留用于调试）
├── edit_main.py                    # 主程序编辑脚本（保留用于调试）
└── electrostatic_field_result.png  # 示例结果图像
```

## 模块说明

### 1. main.py
主程序入口，负责：
- 初始化应用程序窗口
- 协调UI和业务逻辑管理器
- 处理用户交互事件
- 管理视频循环

### 2. ui模块
负责所有用户界面相关的功能：
- **ui_manager.py**: 构建和管理GUI界面，包括侧边栏、首页、实验页、标定页和AI页

### 3. business模块
包含核心业务逻辑：
- **business_manager.py**: 业务逻辑管理器，协调摄像头、标定、数据采集和AI功能
- **cv_tracker.py**: 基于HSV色彩空间的探针标记检测，支持标定和坐标转换
- **calibration.py**: 红色十字标定板检测，计算透视变换矩阵和像素-毫米转换比例
- **data_engine.py**: 电场数据分析引擎，支持数据采集、插值和可视化

### 4. hardware模块
负责硬件设备管理：
- **hw_camera.py**: 摄像头管理，支持多摄像头、帧率控制和图像采集
- **hw_serial.py**: 串口通信管理，支持与硬件设备的连接和数据传输

## 技术特点

1. **模块化设计**: 采用清晰的包结构，分离UI、业务逻辑和硬件管理
2. **多线程处理**: 摄像头采集在独立线程中运行，避免UI卡顿
3. **计算机视觉**: 使用OpenCV进行图像处理、颜色检测和标定
4. **AI集成**: 支持GitHub Models API进行智能分析
5. **数据可视化**: 使用Matplotlib生成电场等势图和梯度图

## 优化改进

1. **代码组织**: 将相关模块归类到对应的包中，提高代码可维护性
2. **性能优化**: 实现多线程摄像头采集，降低CPU占用
3. **识别算法**: 改进颜色检测和标定板识别算法，提高识别准确率
4. **代码复用**: 消除重复代码，提高代码质量

## 依赖项

- Python 3.8+
- customtkinter: 现代化UI框架
- OpenCV: 计算机视觉库
- NumPy: 数值计算库
- PIL/Pillow: 图像处理库
- Matplotlib: 数据可视化库
- SciPy: 科学计算库
- OpenAI: AI API客户端
- pyserial: 串口通信库

## 使用说明

1. 运行主程序：`python main.py`
2. 连接摄像头（可选，支持模拟模式）
3. 在标定页面进行摄像头标定
4. 在实验页面采集数据点
5. 生成电场图并进行AI分析

## 注意事项

- 摄像头索引可能需要根据实际设备调整
- 串口端口需要根据实际硬件配置修改
- AI功能需要有效的GitHub Models API令牌
